\# Hostel Allocation Project



Project hii ni ya \*\*Django\*\* inayofanya allocation ya hostels kwa students.



\## Features

\- Onyesha list ya hostels na students

\- Admin panel kwa ku-manage hostels na students

\- Homepage ya website



\## Tech Stack

\- Python 3.14

\- Django 6.0

\- SQLite (database)
